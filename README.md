# ProScanKill - Windows 进程深度溯源查杀工具

一款功能强大的 Windows 进程深度溯源检测与强制清理工具，专为运维和恶意进程溯源场景设计。

## 项目简介

ProScanKill 是一款专业级的进程分析与清理工具，支持通过 PID、进程名、进程路径三种方式精确定位目标，深度扫描进程关联的系统服务、计划任务和注册表项，并提供强制清理功能。

**核心能力**：
- 🎯 多维度进程定位（PID/进程名/路径）
- 🕵️ 全维度关联检测（服务/计划任务/注册表）
- 💪 强制清理顽固进程及其关联项
- 📊 实时统计汇总与结构化报告输出

---

## 核心功能

### 1. 多维度进程定位

支持三种输入方式，灵活应对不同场景：

| 输入方式 | 适用场景 | 示例 |
|---------|---------|------|
| **PID** | 精确终止单个进程 | `1234` |
| **进程名** | 批量终止同名进程 | `Run_12345.exe` |
| **进程路径** | 精确匹配特定文件 | `C:\ProgramData\Run.exe` |

**批量终止示例**：
```
场景：Run_12345.exe 同时运行在 PID 1234 和 PID 5678
问题：只终止 PID 1234，PID 5678 的进程仍在运行
解决：使用进程名"Run_12345.exe"清理，自动终止所有相关进程
```

### 2. 全维度关联检测

扫描目标进程的所有关联项：
- **进程本体**：获取进程详细信息（PID、路径、命令行、父进程等）
- **系统服务**：检测关联的 Windows 服务
- **计划任务**：扫描相关的计划任务
- **注册表**：检测启动项和关联配置（支持全量扫描）

### 3. 强制清理功能

支持选择性清理操作：
- 🔴 **终止进程**：强制终止匹配的进程
- 🗑️ **删除文件**：删除进程对应的可执行文件
- 📝 **删除注册表项**：清理关联的注册表启动项
- ⏰ **删除计划任务**：移除关联的计划任务

### 4. 智能防护机制

- **守护进程检测**：自动检测并处理守护进程和子进程
- **计划任务监控**：删除后持续监控，防止被重新创建
- **权限检测**：自动检测管理员权限

## 处理流程

```
输入目标 → 扫描进程 → 关联检测 → 生成报告 → 选择性清理
```

**详细步骤**：
1. **输入阶段**：选择 PID、进程名或进程路径输入目标
2. **扫描阶段**：扫描目标进程及其所有关联项
3. **关联检测**：检测系统服务、计划任务、注册表启动项
4. **报告生成**：输出结构化检测报告和统计汇总
5. **清理阶段**：按安全顺序执行清理（注册表 → 计划任务 → 服务 → 进程 → 文件）

## GUI 版本切换

本项目提供两个 GUI 版本：

| 特性 | PyQt6 (`gui.py`) | Tkinter (`gui_tkinter.py`) |
|-----|------------------|----------------------------|
| 界面美观度 | ✅ 更好 | ⚠️ 简单 |
| 兼容性 | Windows 7+ | ✅ Windows XP+ |
| 文件大小 | 较大 | ✅ 较小 |
| 推荐场景 | 现代系统 | 老旧系统兼容 |

### 切换方法

**默认版本**：PyQt6 (`gui.py`)

**切换到 PyQt6**（推荐）：
```python
def main():
    from gui import ProScanKillMainWindow
    from PyQt6.QtWidgets import QApplication
    app = QApplication(sys.argv)
    window = ProScanKillMainWindow()
    window.show()
    sys.exit(app.exec())
```

**切换到 Tkinter**（兼容老旧系统）：
```python
def main():
    from gui_tkinter import ProScanKillApp
    import tkinter as tk
    root = tk.Tk()
    app = ProScanKillApp(root)
    app.run()
```

## 使用方法

### 操作步骤

1. **输入目标**：在 PID、进程名或路径输入框中输入目标信息
2. **检测溯源**：点击"检测溯源"按钮进行全维度扫描
3. **查看结果**：在检测报告区域查看扫描结果和统计汇总
4. **强制清理**：点击"强制清理"按钮，勾选清理选项后确认执行
